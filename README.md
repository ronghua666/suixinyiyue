# 随心一阅 — AI 自动阅卷辅助工具

一款 Windows 桌面 AI 阅卷工具。截图、识别、评分、输入，全自动完成。

无需注册、无需登录，下载即用。自带 API Key，直接调用 AI 服务商。

## 功能特性

- 智能识别答题区 — 框选屏幕区域，AI 自动分析题目数量和每空分值
- 逐题精准评分 — 每道题独立配置标准答案和评分细则，AI 按空评分
- 连续自动批改 — 一份接一份，自动检测翻页
- 无需登录注册 — API 密钥本地存储，你的密钥你做主

## 下载

前往 [官方网站](https://suixinyiyue.xyz) 或 [GitHub Releases](https://github.com/ronghua666/suixinyiyue/releases) 下载最新版本。

支持 Windows 8 / 10 / 11 · x86

## 使用教程

1. 框选区域 — 标记答题区、成绩栏和提交按钮的位置
2. 分析题目 — AI 自动截取答题区，识别题目和空数
3. 配置答案 & 细则 — 填写标准答案和评分细则
4. 开始批改 — 先「试改」验证效果，再「连续批改」自动处理

## 自行构建

```bash
# 安装依赖
pip install -r requirements.txt

# 启动本地服务
python local_server.py

# 浏览器打开 http://127.0.0.1:8766
```
